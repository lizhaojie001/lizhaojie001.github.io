Xcode 8.3.2安装插件
Post：2017-06-11 23:31:57 Tags：/ 插件 / Visit：

转自Xcode8.3.2 使用插件

  1. 先下载原版 Xcode8
  2. 在应用程序中将 Xcode8 复制一份(用来以后打包上传)
  3. 打开 钥匙串访问
  4. 菜单 - 钥匙串访问 - 证书处理 - 创建证书
  • 第一行名称填写 XcodeSigner
  • 第二行不变
  • 第三行选 代码签名
  • 其他的都不变 点击 创建.
  1. 终端运行(要等10多分钟,固态硬盘会快一些)
  1. 安装 Alcatraz
  2. 使用 Alcatraz 安装插件
